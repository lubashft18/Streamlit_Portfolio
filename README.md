📌 STREAMLIT MULTI-PAGE PORTFOLIO (Activity 2)

This project is a Streamlit multi-page portfolio application developed as part of a class activity. It serves as a personal portfolio that showcases my identity, skills, and growth as an aspiring developer.

The application is structured into multiple interactive pages, making it easy to navigate and explore different sections of my profile. It highlights not only my background but also my technical capabilities and completed works.

🔗 Live Demo
👉 View my portfolio here:
https://portfolio-antonioluba.streamlit.app

🔍 Features
•🏠 Home Page – Provides a welcoming introduction and overview of the portfolio

•👤 About Page – Contains my personal information, background, and goals as a developer
•💡 Skills Page – Displays my technical and soft skills
•📜 Certificates Page – Showcases certifications and achievements I have earned
•💻 Projects Page – Highlights the systems and applications I have developed, focusing on real-world and academic projects
•📧 Contact Page – Allows users to reach out or connect with me

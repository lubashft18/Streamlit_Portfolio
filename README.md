# 💻 Streamlit Multi-Page Portfolio (Activity 2)

This project is a **Streamlit multi-page portfolio application** developed as part of a class activity. It serves as my personal portfolio to showcase my identity, skills, certificates, and projects as an aspiring developer.

The application is designed with multiple pages for easy navigation and a better user experience. It highlights not only my background but also my technical capabilities and system projects that I have developed.

---

## 🔗 Live Demo

👉 **View my portfolio here:**  
https://portfolio-antonioluba.streamlit.app/

---

## 🔍 Features

- 🏠 **Home Page**  
  Provides a welcoming introduction and overview of the portfolio  

- 👤 **About Page**  
  Contains my personal information, background, and goals as a developer  

- 💪 **Skills Page**  
  Displays my technical and soft skills  

- 📁 **Projects Page**  
  Highlights the systems and applications I have developed  

- 📧 **Contact Page**  
  Allows users to reach out or connect with me
